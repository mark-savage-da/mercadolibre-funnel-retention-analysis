# MercadoLibre Funnel & Retention Analysis

## Project Overview
This project analyzes user journey performance and retention behavior for MercadoLibre using SQL.

The objective was to identify where users drop off across the conversion funnel, evaluate cohort retention over time, and propose actionable business improvements based on the findings.

## Business Goal
The product team wanted to understand:

- where the biggest user losses occur in the purchase journey
- how conversion changes across key funnel stages
- how retention performs over time for registered users
- what actions could improve growth and retention

## Scope of Analysis
The project covered two main areas:

### 1. Funnel Analysis
The conversion funnel was built using key user events such as:

- first_visit
- select_item / select_promotion
- add_to_cart
- begin_checkout
- add_shipping_info
- add_payment_info
- purchase

The analysis focused on conversion rates between stages, biggest drop-off points, and performance differences by country, device category, and referral source.

### 2. Retention Analysis
User retention was analyzed by cohort using signup_date as the cohort starting point.

Main checkpoints included:

- D7
- D14
- D21
- D28

The analysis also compared retention behavior by country.
